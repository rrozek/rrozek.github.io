    JavaScript SDK Design

    🚧 FUTURE DEVELOPMENT - NOT YET IMPLEMENTED

    This document describes the planned JavaScript/TypeScript SDK design that will wrap the MatterFi C API. The C API export is currently in development, and JavaScript bindings will be implemented in a future phase.

    Current Status

    • ✅ C API Export: Complete with unified result structures
    • 🚧 JavaScript Bindings: Planned for next development phase
    • 🚧 TypeScript SDK: Planned for next development phase

    For current development using the C API directly, see C Bindings Documentation.

    JavaScript bindings for the MatterFi C API using unified result structures.

    Data Structures

    // Unified error codes (mirroring C API)
const ErrorCodes = {
  SUCCESS: 0,
  INTERNAL_ERROR: 1,
  RECIPIENT_INVALID: 2,
  NOT_FOUND: 4,
  ADDRESS_CALCULATING_ERROR: 5,
  ADDRESS_CONVERSION_ERROR: 6,
  INVALID_CLAIM: 7,
  INVALID_ACCOUNT: 8,
  INVALID_SEED: 9,
  INVALID_RECIPIENT: 10,
  INVALID_SFNAME: 11,
  SFNAME_IN_USE: 12,
  INSUFFICIENT_FUNDS: 13,
  FEE_TOO_LOW: 14,
  TRANSFER_FAILED: 15,
  PAYMENT_NOT_FOUND: 16,
  PAYMENT_CANNOT_CANCEL: 17,
  INVALID_TOKEN: 18,
  STAKE_TOO_LOW = 19,
  UNSTAKE_NOT_ALLOWED = 20,
  VALIDATOR_NOT_FOUND = 21
};

// Unified result structure (mirroring C API)
class MatterFiResult {
  constructor(error, data) {
    this.error = error;
    this.data = data;
  }

  isSuccess() {
    return this.error === ErrorCodes.SUCCESS;
  }

  getData() {
    return this.data;
  }

  getErrorMessage() {
    return Object.keys(ErrorCodes).find(key => ErrorCodes[key] === this.error) || 'UNKNOWN_ERROR';
  }
}

    Quick Start

    const MatterFi = require('matterfi-sdk');

// Initialize SDK
await MatterFi.init('mywallet', JSON.stringify({ networks: ['bitcoin'] }));

// Generate seed
const seedResult = MatterFi.seedGenerate();
if (seedResult.isSuccess()) {
  console.log('Recovery words:', seedResult.getData());
} else {
  console.error('Error:', seedResult.getErrorMessage());
}

// Import existing seed
const importResult = MatterFi.seedImport(words, '', 0, 0);
if (!importResult.isSuccess()) {
  console.error('Import failed:', importResult.getErrorMessage());
}

// Create identity
const nymResult = MatterFi.createNym(words, 'password', 'alice');
if (nymResult.isSuccess()) {
  console.log('Identity created successfully');
}

// Create account
const accountResult = MatterFi.makeAccount('bitcoin');
if (accountResult.isSuccess()) {
  const accountId = accountResult.getData();

  // Check balance
  const balanceResult = MatterFi.getBalance(0, accountId); // 0 = bitcoin chain
  if (balanceResult.isSuccess()) {
    console.log('Balance:', balanceResult.getData());
  }

  // Send payment
  const feeResult = MatterFi.calculateFee(0);
  if (feeResult.isSuccess()) {
    const fee = feeResult.getData();
    const paymentResult = MatterFi.sendPayment(0, accountId, 'recipient', '1000000', fee);
    if (paymentResult.isSuccess()) {
      console.log('Payment sent:', paymentResult.getData());
    }
  }
}

// Cleanup
MatterFi.shutdown();

    Chain IDs (SLIP-44 Standard):

    Error Handling

    // Error handling with unified result structure
const seedResult = MatterFi.seedImport("bad words", "", 0, 0);
if (!seedResult.isSuccess()) {
  console.error(`Import failed: ${seedResult.getErrorMessage()}`);
}

// Multiple operation example with proper error handling
async function performWalletOperations() {
  try {
    // Generate seed
    const seedResult = MatterFi.seedGenerate();
    if (!seedResult.isSuccess()) {
      throw new Error(`Seed generation failed: ${seedResult.getErrorMessage()}`);
    }
    const words = seedResult.getData();

    // Import seed
    const importResult = MatterFi.seedImport(words, '', 0, 0);
    if (!importResult.isSuccess()) {
      throw new Error(`Seed import failed: ${importResult.getErrorMessage()}`);
    }

    // Create identity
    const nymResult = MatterFi.createNym(words, 'password', 'alice');
    if (!nymResult.isSuccess()) {
      throw new Error(`Identity creation failed: ${nymResult.getErrorMessage()}`);
    }

    // Create account
    const accountResult = MatterFi.makeAccount('bitcoin');
    if (!accountResult.isSuccess()) {
      throw new Error(`Account creation failed: ${accountResult.getErrorMessage()}`);
    }

    console.log('All operations completed successfully!');
    return accountResult.getData();

  } catch (error) {
    console.error('Wallet operation failed:', error.message);
    return null;
  }
}

    Memory Management

    The SDK handles memory management automatically. JavaScript objects are managed by the JS runtime, while C strings are freed by the native bindings.
